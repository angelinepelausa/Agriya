# Agriya

Agriya is a Filipino mobile platform that connects local farmers directly with consumers. It allows farmers to sell their fresh produce, including fruits, vegetables, grains, and more, to users in their community. The app supports sustainable farming practices and promotes the growth of local agricultural businesses.

## Prerequisites
Make sure you have the following installed:
- Node.js (with npm)

## Git Workflow
1. Clone the Repository (First-Time Setup)
If you're contributing for the first time, clone the repository to your local machine:

git clone https://github.com/angelinepelausa/Agriya.git

Navigate into the project folder:

cd Agriya

2. Pull the Latest Changes from Main
Before making any changes, always ensure your local branch is up to date:

git checkout main  # Switch to the main branch
git pull origin main  # Get the latest updates from the remote repository

3. Create a New Branch for Your Changes
Instead of working directly on main, create a new branch for work:

git checkout -b your-name

Replace your-name with a your name, e.g., Rovic, Fred, Ciarra.

4. Make Changes and Commit
After making your changes, add and commit them:

git add .
git commit -m "Added login screen functionality"

5. Push Your Branch to GitHub
Send your branch to the remote repository:

git push origin your-name

6. Create a Pull Request (PR)
- Go to the GitHub repository.
- Find your pushed branch under Branches.
- Click "Compare & pull request".
- Add a description of your changes.
- Click "Create pull request".
  
7. Wait for Review and Approval
- Your pull request will be reviewed.
- Once approved, it will be merged into main.
  
8. Update Your Local main After Merging
